# River Records

## 题目要求

给定一个整数数组 arr，要求在不改变数组顺序的情况下，找到任意一个元素与它之前出现的所有较小元素之间的最大差值。如果不存在比当前元素小的之前的元素，则返回 -1。

```typescript
const arr = [5, 3, 6, 7, 4];
```

最终的最大差值是 4。

```typescript
const arr = [4, 3, 2, 1];
```

在这个数组中，每个元素之前都没有比它更小的元素，因此返回 -1。

> 在这道题目中，数组中的元素可以类比成水位记录，就像是在观察水流的过程中，记录下水位变化的最大差值一样。
